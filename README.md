# lame-jam-62

This game was made for Lame Jam 62 with the theme Censorship. It is a work simulator, similar to Paper's Please where you have to review the newspaper articles submitted to you. If you fail 3 times (wrong reviews are more than right reviews) you fail.

Controls:
Mouse to drag the papers around

I used Godot 4.6.2 to make this game.
